# Todo App

This is a simple todo app built with React. It allows you to add, edit, and delete todo items.

## Features

- Add new todo items
- Edit existing todo items
- Delete todo items
- Mark todo items as completed

## Technologies Used

- React
- CSS
- Font Awesome
